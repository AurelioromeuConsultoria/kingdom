# 🚀 Quick Start

## Instalação Rápida

### 1. Instalar dependências
```bash
npm install
```

### 2. Configurar API
Crie um arquivo `.env` na raiz do projeto:

```env
VITE_API_BASE_URL=http://localhost:7000/api
```

Substitua pela URL da sua API .NET.

### 3. Executar
```bash
npm run dev
```

O site estará em `http://localhost:3000`

## 📋 Checklist de Configuração

- [ ] Instalar dependências (`npm install`)
- [ ] Criar arquivo `.env` com URL da API
- [ ] Verificar se a API .NET está rodando
- [ ] Testar endpoints da API
- [ ] Adicionar imagens em `public/images/` (logo, banners, etc)

## 🎯 Próximos Passos

1. **Configurar API .NET**: Certifique-se de que os endpoints estão implementados
2. **Adicionar Imagens**: Coloque logo, banners e outras imagens em `public/images/`
3. **Personalizar Conteúdo**: Edite os componentes em `src/pages/` e `src/components/`
4. **Testar Integração**: Verifique se os dados estão sendo carregados corretamente
